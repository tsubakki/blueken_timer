# ブル研タイマー
北大ブルーグラス研究会におけるライブ用の特殊タイマー

# 機能
- オンラインタイマー機能
  - オンラインで時間が同期され、各スマホでリアルタイムで状況がわかる
- PA用の赤・青・緑の信号
  - 赤：「配線などPAシステムに問題がある警告」の信号
  - 青：「問題がないため演奏可能」の信号
  - 緑：「問題が発生したため現在対応中」の信号
